# CSS Reset UdyUX

> My evolving CSS reset, drawing some inspiration from reset.css and normalize among other projects.

## Usage

### Install

```bash
npm install css-reset-udyux
# or
yarn add css-reset-udyux
```

### Import

```scss
// in sass
@import '~css-reset-udyux/reset', 'my-styles/come-after';
```
